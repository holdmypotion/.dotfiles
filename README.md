# Dotfiles

My personal dotfiles.

## Installation

```bash
git clone https://github.com/rahulsharma/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
```

## Structure

* `.zshrc`: Zsh configuration
* `.tmux.conf`: Tmux configuration
* `.config/`: Application configurations
